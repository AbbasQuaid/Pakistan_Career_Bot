# 🎓 Pakistan Career Counseling Bot

An **AI-inspired Career Counseling System for Pakistani Students** that recommends the most suitable academic stream based on students’ **interests, personality traits, and career preferences**.

This project is built using **Python and Streamlit** and provides an interactive interface where students can answer questions and receive **career guidance along with personality analysis**.

---

# 📌 Project Overview

Choosing the right career path is one of the most important decisions for students after matriculation in Pakistan. Many students are confused between streams, such as:

* FSc Pre-Medical
* FSc Pre-Engineering
* ICS (Computer Science)
* ICom (Commerce)

This system helps students identify the most suitable academic stream based on:

* Academic interests
* Career preferences
* Personality traits (MBTI Model)

The system then provides a **recommended stream and personality analysis**.

---

# 🚀 Features

### 🎯 Career Stream Recommendation

The system recommends one of the following academic paths:

* 🧪 **FSc Pre-Medical**
* ⚙ **FSc Pre-Engineering**
* 💻 **ICS (Computer Science)**
* 💼 **ICom (Commerce)**

---

### 🧠 Personality Assessment (MBTI Model)

The chatbot analyzes personality traits using the **MBTI framework**, which includes:

* **Introversion (I) vs Extroversion (E)**
* **Thinking (T) vs Feeling (F)**
* **Sensing (S) vs Intuition (N)**
* **Judging (J) vs Perceiving (P)**

Example results:

* INTJ
* ENFP
* ISTP
* ENTJ

---

### 📊 Personality Radar Chart

The system visualizes personality traits using a **Radar Chart**, helping students better understand their behavioral tendencies.

---

### 🔐 Admin Dashboard

The system also includes a **password-protected admin panel** that allows administrators to:

* View all student counseling records
* Analyze recommended streams
* Monitor student data
* View analytics charts
